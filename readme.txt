Quickly set links to Bee English Dictionary definitions of words/phrases containing meaning, audio, phonetics, usages and other informations.

== Description ==

Link to Dictionary Meaning allows you to quickly set links for difficult words to Bee English Dictionary definitions of words/phrases containing meaning, audio, phonetics, usages and other informations.

*Write "{bd:polemic}" to create a link to the english lemma of "polemic".

== Installation ==

1. Upload the plugin package to your wp-content/plugins directory
1. Unzip the package (unzip lexidef.zip)
1. Go to your "plugins" administration page and activate the "Link to Dictionary Meaning" plugin
